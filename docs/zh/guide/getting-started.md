## Claude Code 是什么？

Claude Code 是 Anthropic 公司开发的人工智能助手，专门用于帮助软件开发者和研究人员完成编程任务。与传统的聊天机器人不同，Claude Code 可以：

- **直接操作你的文件**：读取、编辑、创建项目中的任何文件
- **运行代码**：执行 Python、R、MATLAB 等代码并查看结果
- **管理项目**：搜索代码、理解项目结构、协助调试
- **使用专业工具**：调用数百个专门的技能（skills）完成复杂任务

### 与普通 AI 聊天的区别

| 普通聊天 AI | Claude Code |
|-------------|-------------|
| 只能对话，不能操作文件 | 可以直接读写你的文件 |
| 代码需要你复制粘贴 | 自动写入文件并运行 |
| 不了解你的项目 | 理解整个项目结构 |
| 通用性回答 | 针对你的代码给出建议 |

## 安装与配置

### 安装步骤

1. **访问官网**：前往 [Claude Code 官网](https://claude.com/claude-code)
2. **下载安装包**：根据你的操作系统（Windows/Mac/Linux）下载对应版本
3. **安装**：运行安装程序，按提示完成安装
4. **登录**：首次运行需要登录 Anthropic 账号

### 验证安装

打开命令行（终端），输入：

```bash
claude --version
```

如果显示版本号，说明安装成功。

## 基础操作

### 启动 Claude Code

在命令行中输入：

```bash
claude
```

或者在 VSCode 中打开 Claude Code 扩展。

### 第一个对话

启动后，你可以用自然语言与 Claude 对话：

```
你好！我是一名水文环境研究人员
```

Claude 会了解你的背景，并在后续对话中提供更相关的帮助。

### 让 Claude 做事

Claude 最强大的地方是可以**执行任务**，而不只是回答问题。例如：

```
请帮我创建一个 Python 脚本，读取水文数据 CSV 文件
```

Claude 会：
1. 理解你的需求
2. 创建相应的 Python 文件
3. 写入代码
4. 告诉你如何使用

## 对话技巧

### 1. 明确你的目标

好的提问：
```
帮我分析 rainfall.csv 文件，计算平均降雨量和最大降雨量
```

不够好的提问：
```
看看这个数据
```

### 2. 提供上下文

告诉 Claude 更多信息，它会给出更好的答案：

```
我有一个水文站的数据文件，包含日期、水位和流量三列。
请帮我画出水位-流量关系图，数据文件名是 station_001.csv
```

### 3. 分步骤描述复杂任务

对于复杂任务，可以分步骤描述：

```
我想做以下分析：
1. 读取水文数据文件
2. 筛选出汛期（6-9月）的数据
3. 计算汛期平均流量
4. 画出时间序列图
```

### 4. 使用具体文件名

当提到文件时，尽量使用完整的文件名或路径：

```
分析 C:\data\rainfall_2023.csv 文件
```

## 项目工作区

### 在项目中使用 Claude Code

建议将每个研究项目放在独立的文件夹中，然后在项目目录下启动 Claude Code：

```bash
cd C:\Users\你的用户名\research\hydrology_project
claude
```

这样 Claude 可以：
- 看到项目中的所有文件
- 理解项目结构
- 在正确的位置创建新文件

### 常用命令

在 Claude Code 中，你可以使用一些特殊命令：

- `/clear` - 清除对话历史
- `/help` - 显示帮助信息
- `/exit` - 退出 Claude Code

## 常见问题

### Q: Claude 能读取我的所有文件吗？

A: 默认情况下，Claude 只能访问你明确告诉它的文件。但如果你在项目目录中启动，它可以查看项目中的所有文件。

### Q: 我的代码会被发送到云端吗？

A: 是的，Claude 需要将代码发送到 Anthropic 的服务器进行分析。请确保遵守你所在机构的数据安全政策。

### Q: Claude 支持哪些编程语言？

A: Claude 支持几乎所有主流编程语言，包括 Python、R、MATLAB、Julia、JavaScript 等。水文研究中常用的 Python 和 R 都有很好的支持。

### Q: Claude 会直接修改我的文件吗？

A: Claude 会先告诉你它计划做什么，等待你确认后才会修改文件。你始终掌控着你的代码。

### Q: 如果 Claude 给的代码有错误怎么办？

A: 直接告诉 Claude 它报错了，把错误信息粘贴给它，它会帮你调试和修复。
