# Claude Code 项目配置

## 项目说明

这是一个水文环境研究人员的 Claude Code 使用指南项目。

## 目标用户

- 水文环境领域的研究人员和研究生
- 需要处理水文数据、进行统计分析的研究者
- 希望借助 AI 提升研究效率的科研工作者

## 语言规范

### 绝对强制使用简体中文
- 所有 AI 回复、文档、注释、日志、提交信息等必须强制使用简体中文
- 唯一例外：代码标识符（变量名、函数名、类名等）遵循项目既有命名约定（通常使用英文）

## 编码规范

### Python 代码
- 使用 UTF-8 无 BOM 编码
- 遵循 PEP 8 代码风格
- 函数必须有文档字符串（docstring）
- 变量名使用英文，注释使用中文

### Markdown 文档
- 使用中文编写内容
- 代码块中的代码使用英文标识符
- 链接和引用使用相对路径

## 项目结构

```
CCusage/
├── README.md           # 中文版首页
├── README-en.md        # 英文版首页
├── CLAUDE.md           # AI 助手配置（本文件）
├── claude.md           # 项目级配置（同步内容）
├── docs/               # 详细文档目录
│   ├── 01-基础入门.md          # 中文版
│   ├── 01-Getting-Started.md   # 英文版
│   ├── 02-常用功能.md          # 中文版
│   ├── 02-Common-Features.md   # 英文版
│   ├── 03-水文场景案例.md      # 中文版
│   ├── 03-Hydrology-Use-Cases.md  # 英文版
│   ├── 04-进阶技巧.md          # 中文版
│   └── 04-Advanced-Tips.md     # 英文版
└── examples/           # 示例代码和数据
    ├── data/           # 示例数据文件
    └── scripts/        # 示例脚本
```

## 数据规范

### 示例数据文件
- 所有示例数据文件放在 `examples/data/` 目录
- CSV 文件使用 UTF-8 编码
- 日期格式建议使用 ISO 8601（YYYY-MM-DD）

### 示例脚本
- 所有示例脚本放在 `examples/scripts/` 目录
- 脚本必须包含完整的中文注释
- 脚本必须包含使用示例

### 数据处理
- `pandas`：数据处理和分析

### 可视化
- `matplotlib`：基础绘图

## 文档维护规则

### 双语同步
- 每个文档都有中文版和英文版
- 修改任何内容时，必须同步更新两个语言版本
- 文件命名约定：
  - 中文版：`docs/XX-中文名称.md`
  - 英文版：`docs/XX-English-Name.md`

### 相互引用
- 中文版文档引用中文版文档
- 英文版文档引用英文版文档
- 所有文档顶部包含语言切换链接
- 所有文档底部包含导航链接

### 文档更新检查清单
修改任何文档后，检查是否需要更新：
- [ ] 对应语言版本的文档
- [ ] README.md 或 README-en.md
- [ ] 文档之间的相互引用链接
- [ ] 目录和导航

## 代码示例规范

### 完整性
- 代码示例必须完整可运行
- 包含必要的导入语句
- 包含数据生成或加载代码
- 包含结果输出或可视化代码

### 注释规范
```python
# 单行注释：说明代码功能

"""
函数/类的文档字符串
包含：功能描述、参数说明、返回值说明
"""

def example_function(param1, param2):
    """
    函数功能描述

    参数:
        param1: 参数1说明
        param2: 参数2说明

    返回:
        返回值说明
    """
    pass
```

### 示例数据
- 优先使用模拟数据，确保代码可直接运行
- 如需真实数据，提供数据格式说明
- 数据文件需包含在项目中或说明获取方式

## 技能使用偏好

### 优先使用的技能
对于水文研究场景，优先使用以下技能：
- `csv-data-summarizer`：CSV 数据快速分析
- `scientific-visualization`：科研级图表生成
- `statistical-analysis`：统计分析方法选择和执行
- `citation-management`：文献引用管理
- `scientific-writing`：科研写作辅助

### 技能调用方式
```
请使用 [技能名] 帮我 [具体任务]
```

例如：
```
请使用 csv-data-summarizer 分析 rainfall.csv 文件
```

## 特别注意事项

1. **水文领域特殊性**
   - 重视数据的时间序列特性
   - 注意物理单位的正确性
   - 考虑水文现象的季节性和周期性
   - 关注极值事件（洪水、干旱）的分析

2. **中文文档优先**
   - 主要面向中文读者
   - 示例和说明使用中文
   - 英文版保持同步更新

3. **实用性导向**
   - 所有内容来自实际应用场景
   - 提供完整可用的解决方案
   - 避免过度理论化

4. **可维护性**
   - 保持文档结构清晰
   - 使用相对路径链接
   - 双语版本同步更新
